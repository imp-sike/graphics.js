Introduction Last updated: Today

We often encounter a lot of problems while setting up graphics.h header files with C and we do not want to do a lot of stuffs just to learn simple graphics algorithm. Graphics.h is an online alternative to graphics.h that is totally recreated using HTML5 canvas and work with bare minimum knowledge of js.

Basic Syntax

Like each C program, Graphics.js program also starts with main function. The main function, unlike C, must be assigned async. The bare minimum code for a sample line drawing Graphics.js program is as below: 

// async main function
async function main() {
	initgraph();
	line(100,200,600,800);
	closegraph();
}
main() // at the end of program the main function must be called so that it gets executed.

The arc() function

Declaration: void arc(int x, int y, int stangle, int endangle, int radius);
"arc" function is used to draw an arc with center (x, y) and stangle specifies starting angle, endangle specifies the end angle and last parameter specifies the radius of the arc. arc function can also be used to draw a circle but for that starting angle and end angle should be 0 and 360 respectively. 

arc(100, 75, 50, 0, 2 * Math.PI);

The circle() function

circle(x, y, radius);

The cleardevice() function

Used to clear the graphics canvas.

cleardevice();

The ellipse() function

Used to draw ellipse.

ellipse(x,y,sangle,endangle,xradius,yradius);

The putpixel() function

Like C, We have to provide the x and y coordinate position following with a color to putpixel(). 

putpixel(xCoordinate,yCoordinate,"#ff0000");

The outtextxy(x,y,string) function

Used to output a string at a given x,y coordinate. 

outtextxy(xCoordinate,yCoordinate,"Hello World!");

Helper Function Last updated: Today

Helper functions are not a part of Graphics.js pure graphic functions rather the functions that helps in better visualizations of the graphics. Currently supported functions are delay() and abs() functions

The delay() function

The delay() function is used to suspend execution of a program for a particular time. 

// delay function example
async function main() {
	outtextxy(200, 20, "Printed before 50 milliseconds.");
	await delay(50); 
	outtextxy(200, 20, "Printed after 50 milliseconds.");
}

The abs() function

The abs() function is used to get the absolute value of a number. abs() function returns the absolute value of a number. That is, it returns x if x is positive or zero, and the negation of x if x is negative. 

// abs function example
var x1 = abs(-5);
outtextxy(200, 20, x1); // 5

The input() function

It is used to get input from the user. This function returns a string. 

// abs function example
var x1 = input("Enter the number");

The print() function

The print() function is used to output a string to the canvas. 

// abs function example
print("Hello World")
